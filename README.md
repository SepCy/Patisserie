# Patisserie

## Comment utiliser ?
### Prerequis
1. php 7.*
2. composer

### Exécution
1. Cloner le repo
2. se déplacer dans le repertoire
3. ouvrir un terminal puis taper "composer install" puis "npm run-script build"
4. Enfin lancer le serveur en tapant la commande "php -S localhost:8000" -t ./public
